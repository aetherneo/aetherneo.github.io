# AetherGhost 产品定义文档 (PRD) v3.0

### 1. 核心产品逻辑 (Product Logic)

* **Web-Centric (全网页化)**：无需下载安装包，用户直接通过浏览器登录 AetherGhost 门户，即可访问隔离的云端办公环境。
* **Exclusive Only (纯专属制)**：不提供任何共享资源。每个号码、每个环境都是用户独占的物理隔离个体，确保极高的账号安全性。

### 2. 三大核心模块 (Core Modules)

* **Ghost-Space (云端网页工作空间)**
* **定义**：在网页内嵌入的隔离工作容器。
* **特性**：通过服务端渲染或容器化技术，让用户在同一个浏览器中开启多个互不干扰的办公标签页。


* **Phantom-Line (专属身份线)**
* **定义**：纯订阅制的个人专属通信端口。
* **特性**：**不设共享池**。用户租用的每一个号码都是从底层运营商直接分配的干净长号，仅供该用户接收 2FA 验证码。


* **Vortex-Stream (加密数据流)**
* **定义**：内置于网页空间的加密传输协议。
* **特性**：确保用户在云空间内的一切操作均通过企业级加密隧道中转。

### 4. 商业计划书 (合规增强版)

#### **一、 商业模式：身份即服务 (Identity as a Service)**

VortexNode/AetherGhost 不提供临时的“接码服务”，我们提供的是**数字身份的长期维护方案**。通过将“号码、空间、网络”绑定在同一个半年度订阅周期内，我们解决了跨境业务中“环境不稳定”和“身份易丢失”的痛点。

#### **二、 合规性防御 (Compliance Defenses)**

* **非代理化声明**：我们不销售“翻墙工具”，我们销售的是“具有安全连接能力的云端办公空间”。
* **独占性证明**：由于不设共享池，我们完美规避了“号码被多人重复使用进行违法活动”的风险。每个号码都有明确的订阅记录。
* **金融风控**：系统内置金融关键词过滤，主动拦截任何试图将号码用于非法金融开户的行为。

#### **三、 盈利预测**

* **成本**：Telnyx 号码成本约 $1.1/月 + 极低比例的 Web 托管成本。
* **毛利**：以 Pro 套餐为例，半年收入 $29，成本约 $7-$9，**毛利空间在 70% 左右**。
